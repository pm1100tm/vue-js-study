데이터 바인딩

데이터 바인딩의 가장 기본적인 형태는 Mustache 구문임 {{}} (이중 괄호)

메시지: {{ msg }}

Mustache 태그는 해당 데이터 객체의 msg 속성 값으로 대체.
데이터 객체의 msg 속성이 변경될 때 마다 갱신됨.

v-once

v-once 를 사용하여 데이터 변경 시 업데이트 되지 않는 일회성 보간 수행.
그러나 같은 노드의 바인딩에도 영향을 끼침

다시는 변경하지 않습니다: {{ msg }} 음 역시 아래꺼는 변경 된다.
하지만 같은 노드의 여기서는 ? {{ test }}
이거는 변경이 되려나 같은 노드 아닌 거 같은데 {{ msg2 }} 어 된다

이중 괄호는 HTML이 아닌 일반 텍스트로 이해함.
실제 HTML 출력하려면 v-html 을 사용해야 함.

Using mustaches: {{ rawHtml }}

JavaScript 표현식 사용하기

지금까지 템플릿의 간단한 속성 키에만 바인딩했습니다.
그러나 실제로 Vue.js는 모든 데이터 바인딩 내에서 JavaScript 표현식의 모든 기능을 지원합니다.

{{ number + 1 }}
{{ ok ? 'YES' : 'NO' }}
{{ message.split('').reverse().join('') }}

표현식은 Vue 인스턴스 데이터 범위 내에서 JavaScript로 계산.
제한사항은 각 바인딩에 하나의 단일 표현식 만 포함될 수 있다.
조건문은 작동하지 않는다. 삼항 연산자를 사용해야 한다.
